Standard:

```jsx
<Input label="Label" placeholder="Placeholder" />
```

With Icon:

```jsx
const icon = <div>✊🏾</div>;
<Input icon={icon} iconPosition="left" />;
```

Disabled:

```jsx padded
<Input label="Label" placeholder="Placeholder" disabled />
```

Error State:

```jsx
<Input label="Label" errorMessage="There is an error" />
```
